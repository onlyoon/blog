# Hello World! 👋

Astro 블로그의 첫 번째 포스트입니다.

## 왜 Astro를 선택했나요?

Astro는 다음과 같은 장점들이 있습니다:

### 🚀 성능

- **Island Architecture**: 필요한 부분만 JavaScript로 변환
- **Zero JS by default**: 기본적으로 JavaScript가 필요 없음

### 🎨 유연성

- React, Vue, Svelte 등 다양한 프레임워크와 함께 사용 가능
- 마크다운 기반 콘텐츠 관리

### 📝 개발 경험

- TypeScript 지원
- Hot reload
- 간단한 설정
